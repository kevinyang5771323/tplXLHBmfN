# 项目概述 python781

本项目是一款基于Python语言的图书馆管理系统，使用Django框架进行开发，搭配MySQL数据库和Vue前端框架。

## 技术栈

- Python 3.8
- Django
- MySQL 5.7
- Vue.js

## 功能模块

### 系统角色

#### 管理员

- 首页浏览
- 个人中心
  - 用户管理
  - 图书馆管理
  - 图书信息管理
  - 图书类型管理
  - 借阅信息管理
  - 归还信息管理
  - 续借信息管理
  - 罚金信息管理
  - 留言板管理
  - 系统管理

#### 用户

- 首页浏览
- 个人中心
  - 借阅信息管理
  - 归还信息管理
  - 续借信息管理
  - 罚金信息管理
  - 留言板管理
  - 我的收藏管理

### 系统功能

- 首页：展示系统概况、通知公告等。
- 图书馆：查看图书馆基础信息及藏书情况。
- 公告信息：查看和管理系统公告。
- 留言板：用户与管理员之间的交流平台。

## 运行环境

- Python 3.8及以上版本
- Django框架
- MySQL数据库版本5.7
- Vue.js环境

## 部署步骤

（具体部署步骤请根据实际环境调整，以下仅为概述）

1. 安装Python 3.8及其以上版本
2. 安装Django并根据需求进行配置
3. 安装MySQL 5.7并创建数据库
4. 克隆项目代码并进入项目目录
5. 执行数据库迁移命令以创建数据库表
6. 运行开发服务器或部署到生产环境

## 目录结构

```
project/
│
├── app/              # Django应用目录
│   ├── migrations/   # 数据库迁移文件目录
│   ├── models.py     # 数据模型定义
│   ├── views.py      # 视图处理逻辑
│   └── ...
│
├── static/           # 静态资源文件
├── templates/        # 前端模板文件

└── ...
```
